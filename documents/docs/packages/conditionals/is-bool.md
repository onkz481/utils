The `isBool` method determines if the passed value is `Boolean`.

```js
isBool(true); // true
isBool(false); // true

isBool(0); // false
isBool(null); // false
isBool(undefined); // false
```

## Usage

```js
import { isBool } from '@onkz481/conditionals';

isBool(value);
```
