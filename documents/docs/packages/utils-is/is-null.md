The `isNull` method determines if the passed value is `null` or `undefined`.

```js
isNull(null); // true
isNull(undefined); // true

isNull(''); // false
isNull([]); // false
```

## Usage

```js
import { isNull } from '@onkz481/conditionals';

isNull(value);
```
